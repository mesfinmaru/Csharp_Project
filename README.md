# Car Rental Management System
A desktop application built using C# Windows Forms for managing car rental operations, including vehicle inventory tracking, customer management, and booking reservations.

# Technology Stack
- Language: C#
- Platform: Windows Forms (WinForms)
- Database: MS SQL Server
- IDE: Visual Studio 2022
